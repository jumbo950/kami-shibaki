# 上柴木あぐろ Standard Outfit No Jacket

> このファイルは、上柴木あぐろのStandard Outfitにおけるジャケット非着用・手持ち状態の派生マスター仕様です。  
> 本仕様は独立衣装ではなく、`Standard_Outfit.md` のVariantとして扱います。

## Purpose

Standard_Outfit_No_Jacketは、Standard Outfitにおいてジャケットを脱いだ状態を安定して再現するための仕様書である。

Business Outfitの室内・退勤・移動シーンなどで使用する。

## Canonical State

以下をCanonical No-Jacket Outfitとする。

- ジャケットを着用しない
- ボルドーのノースリーブリブハイネックニット
- ベージュスラックス
- 黒ベルト
- 社員証ストラップ
- ルビーアクセサリー
- 黒パンプス（赤い靴底）

## Jacket Carrying Rule

ジャケットは以下の状態で扱う。

- 片腕に掛ける
- 手に自然に持つ
- 室内
- 退勤
- 移動シーン

ジャケットを持たない状態もNo Jacket Variantとして成立する。

## Exclusive Rule

Jacket Wearing と Jacket Carrying は排他的である。

以下は禁止。

- ジャケットを着用したまま手に持つ
- ジャケットを肩へ掛けたまま着用する
- ジャケットを背負う
- 着用と手持ちを同時に表現する

常に Wearing または Carrying / No Jacket のどちらか一方のみとする。

### Type

- ノースリーブ
- リブ編み
- ハイネック
- 身体へ自然にフィット
- 着丈はパンツ内へ収まる

### Color

- 深いボルドー
- ワインレッド系
- Burgundy: #7A1F2D

### Fit

- Body_Masterのシルエットを直接反映する
- バスト位置を自然に見せる
- ウエストのくびれを自然に見せる
- 胸だけを強調しすぎない

## Pants

- ベージュスラックス
- センタープレス
- ハイウエスト気味
- ベルトループあり
- 太もものラインを自然に拾う
- 裾へ向かって細くなる
- スキニーほど細くしない

## Belt

- ブラックレザー
- ゴールドバックル
- 幅は標準

## ID Card

社員証はNo Jacket Variantでも維持する。

- ボルドーの首掛けストラップ
- 細めのストラップ
- 胸中央付近に自然に垂れる
- ニット上に直接重なる

## Accessories

以下を維持する。

- 赤系アンダーフレーム眼鏡
- ルビーピアス
- ルビーネックレス

## Shoes

- ブラックのポインテッドトゥパンプス
- エナメル調
- 細めヒール
- 赤いアウトソール

## Body Visibility

No Jacket VariantはBody_Masterのシルエットが最も見えやすい状態である。

以下を自然に表現する。

- バスト位置
- ウエスト位置
- ヒップ位置
- 太ももの存在感
- 肩から腕の自然なライン

衣装によって体型が変化して見えてはならない。

## Multi-angle Rule

以下すべての角度で衣装構成を維持する。

- 正面
- 側面
- 背面

以下の位置・形状が変化しないこと。

- ニットの首元
- ノースリーブの肩口
- ベルト位置
- パンツシルエット
- 社員証位置
- ジャケットを持つ場合の腕位置

## NG

- ジャケットを着る
- ジャケットを着用したまま手に持つ
- ジャケットを肩へ掛けたまま着用する
- ジャケットを背負う
- 長袖インナー
- 白シャツへ変更
- スカート化
- 社員証なし
- ベルトなし
- パンプス以外
- Body_Masterを潰すシルエット
- 胸だけを過度に強調する
- 肩幅を変える

## Generation Rule

Standard Outfitでジャケットを脱ぐ場合は、本ファイルを参照する。

ジャケットの状態は必ず以下のどちらか一方にする。

1. `Standard_Outfit.md` の Jacket Wearing
2. 本ファイルの Jacket Carrying / No Jacket

両方を同時に表現しない。
